MCT1

A Minecraft plugin simulating Type 1 Diabetes in Virtual Reality

Github

Taking medical learning out of the classroom

Our aim is simple: to allow newly-diagnosed Type 1 diabetics to learn how to manage their condition in a fun, safe environment. MCT1 is an open-source project that modifies Minecraft. Play as a character with Type 1 diabetes. Go on quests. Be even more awesome.

Fun and engaging

Experience a fun and engaging Minecraft environment: wield magical superpowers and manage T1 diabetes in a sprawling universe.
